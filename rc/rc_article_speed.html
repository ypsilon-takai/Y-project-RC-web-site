飛行速度について考える

YpsilonはSG2がベースで軽量・低翼面荷重なので、このままでは強風時の性能あまりよくありません。っていうか、風が強いとあまり前に進みません。
対策として、

  • バラストを積む
  • 翼弦長を減らす

というのが考えられます。

翼弦長と飛行速度

去年(2006)の夏あたりに、USで翼弦の短い機体をOlegさんらが試験しているという話題がでました。Drela氏も、SG3を設計するなら翼弦を短くすることは考慮に値すると発言していたりして、翼弦の短い機体に興味を持ちました。翼弦を減らすと、飛行時の抵抗が減るため、風に強くなります。また、抵抗が減ることでランチでより高くあがることが期待できます。

Martin Hepperle氏のサイトに、レイノルズ数(Re)の計算式が載っています。

       Re = v * l * 70000

       * Re : レイノルズ数
         v  : 速度
         l  : 翼弦

この式は、レイノルズ数の定義の式に空気の密度などの定数とかを入れ込んで、定数をおおよその値に丸めて得られるものです。

この式はレイノルズ数を求めるための式ですが、翼弦とレイノルズ数がわかっていれば、速度が出ます。SG2は、図面によると、翼根でReは52000、翼弦は7.2in=183mm=0.183mですので、 

       v = Re / l / 70000
         = 52000 / 0.183 / 70000
         = 4.05 m/s

となり、だいたい4m/sの速度を想定していることがわかります。実測したことはないのですが、納得できる値です。
ここでの速度は、当然対気速度ですので、3m/sの風が吹いているときは、向かい風では地面に対して1m/s、追い風では7m/sの速度になります。また、4m/s以上の風が出ている場合は、前に出ないことになります。実際は、迎角を減らす=ダウンを打つなどして前に出すことになるわけです。が、最良の滑空比での飛行とはならず、性能は落ちます。

上記の結果を使って、逆に目的の風速で最適になるような翼弦を求めることができます。
たとえば、同じ翼型をつかって同じReで6m/sで飛行する場合の翼弦の長さは、 

       l = Re / v / 70000
         = 52000 / 6 / 70000
         = 0.124 m = 124mm

になります。ちょっと現実的ではありませんが、そういうことです。

逆に翼弦を決めて速度を出してみると、たとえばSG2の10%引きだとだいたい160mmですので、 

       v = 52000 / 0.160 / 70000
         = 4.6 m/s

が最適速度ということになります。

機体の重量と飛行速度

バラストを積むと、飛行速度が上がるのは経験でわかりますが、理論的にはどうなのでしょう。

最良の揚抗比は翼型とReで決まります。機体の重量は、揚抗比に影響しないので、無風時の機体の最良の飛行経路は機体の重量にかかわらず、同じということになります。
ところが、重くなってるわけですから、同じ速度では揚力が足りませんので、 速度が上がるわけです。(図１)


ちゃんと計算してみましょう。先にあげたリンク先のページに lift coefficient(=揚力係数)の式があります。これを変形して速度と揚力の関係の 式にしてみます。 

      元の式:
      Cl(揚力係数) = 揚力 / ( 空気密度の1/2 * 速度の２乗 * 翼面積)

      変形すると:
      速度の２乗 = 定数 * 揚力
         定数 = １ / ( 空気密度の1/2 * 翼面積 * 揚力係数 )
      速度 = ( 定数 * 揚力 ) の平方根

という式ができます。必要な揚力は重量に比例するわけですから、速度は重量の平方根に比例することになります。

たとえば、250gの機体に50gのバラストを積むと、重量は1.2倍になるので平方根を とって速度は約1.1倍となります。
先に計算したSG2であれば、重量を20%増やすと速度は4m/sの1.1倍の4.4m/sになることを になります。SG2の重さは220g程度ですので、20%は44gです。

最初に計算したときは、速度の差があまり大きくないのが不思議でしたが、とりあえずは気にしないことにしています。でも、なんか、大きな勘違いがあるような気もします。

まとめ

上記の結果を元にして、こんなことを考えました。

翼弦が160mmのYpsilonを作ったとします。このときの最適速度は、前に書いたように、4.6m/sです。
今のYpsilon(翼弦190mm)にバラストを積んで最適速度を4.6m/sにするには、 

      4.6 / 4.0 = 1.15 :: 速度は1.15倍
      この速度にするために重量は 1.15の2乗 = 1.32倍 にする必要がある。
      Ypsilonは240gなので、 240 * 0.32 = 77g

となり、約80gのバラストを積む必要があることになります。

80gバラストの190mmとバラストなしの160mmが同じ性能だとすると、こんな構成にするとかなりの範囲に対応できそうです。

弱風用4.3m190mm 40g
